Сборка проекта:
./gradlew clean test shadowJar

Перейти в cd build/libs/

Запуск:
java -jar phone-okved-matcher.jar 60000
где 60000 - время в миллисекундах до следующей загрузки okved.json из репозитория (если не указывать, то 1 день)

Библиотека jackson, используемая для работы с json распространяется по лицензии Apache 2.0
Библиотека commons-lang3, использованнвя для класса Pair распространяется по лицензии Apache 2.0

Основные условия Apache 2.0:
- Коммерческое использование разрешено без ограничений
- Модификация и распространение кода допустимо
- Требуется сохранение оригинального уведомления об авторских правах и лицензии в копиях
- Нет copyleft требований (не заставляет раскрывать ваш исходный код)

Стандарт документации Oracle/Sun
